Tamil Nadu Election Dashboard – Excel VBA
Project Overview
This project is a professional Excel VBA dashboard built for Tamil Nadu election data analysis.
The dashboard automatically processes raw election data and generates interactive analytics, KPI cards, charts, and winner insights inside Excel.

Features


Automated dashboard generation using VBA


Party-wise vote analysis


Winning party detection


Top candidate identification


KPI cards and analytics


Dynamic chart creation


Automatic calculations


Modern dashboard UI in Excel


Conditional formatting


Professional reporting layout

How to Run
Step 1
Open the .xlsm file in Microsoft Excel.
Step 2
Enable Macros.
Step 3
Press:
ALT + F11
Step 4
Insert a new VBA Module and paste the VBA code.
Step 5
Run:
